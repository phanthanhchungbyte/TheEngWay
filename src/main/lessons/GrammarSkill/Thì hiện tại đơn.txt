<h4 id="Động_từ_thường" style="padding-left: 40px;"><span style="font-family: helvetica, arial, sans-serif;">a) Động từ thường</span></h4>
<div style="padding-left: 40px;">
<ul>
<li style="list-style-type: none;">
<ul>
<li style="font-family: helvetica, arial, sans-serif;"><span style="font-family: helvetica, arial, sans-serif;">Khẳng định (+):&nbsp;<span style="color: #e03e2d;"><strong>S + V (s/es) .</strong></span></span></li>
<li style="font-family: helvetica, arial, sans-serif;"><span style="font-family: helvetica, arial, sans-serif;">Phủ định (-):&nbsp;<span style="color: #e03e2d;"><strong>S + do/does + not + V.</strong></span></span></li>
<li style="font-family: helvetica, arial, sans-serif;"><span style="font-family: helvetica, arial, sans-serif;">Nghi vấn (?):</span>
<ul>
<li style="font-family: helvetica, arial, sans-serif;"><span style="font-family: helvetica, arial, sans-serif;">C&acirc;u hỏi Yes/No question:&nbsp;<span style="color: #e03e2d;"><strong>Do/Does (not) + S + V + O?</strong></span></span></li>
<li style="font-family: helvetica, arial, sans-serif;"><span style="font-family: helvetica, arial, sans-serif;">C&acirc;u hỏi WH- / H- question:&nbsp;<span style="color: #e03e2d;"><strong>WH- word / H- word + do/does (not) + S + V + O?</strong></span></span></li>
</ul>
</li>
</ul>
</li>
</ul>

<li style="font-family: helvetica, arial, sans-serif;"><span style="font-family: helvetica, arial, sans-serif;">Phủ định (-):&nbsp;<span style="color: #e03e2d;"><strong>S + am/is/are + not + N/Adj + O.</strong></span></span></li>

<li style="font-family: helvetica, arial, sans-serif;"><span style="font-family: helvetica, arial, sans-serif;">C&acirc;u hỏi WH- / H- question:&nbsp;<span style="color: #e03e2d;"><strong>WH- word / H- word+ am/is/are + S + N/Adj + O?</strong></span></span></li>

</ul>
